### 4. Test the System

#### As a Customer:
1. Visit http://localhost:3000
2. Click "Enter Menu"
3. Browse menu items
4. Add items to cart
5. Enter table number (any number)
6. Place order
7. Watch order status update in real-time!

#### As an Admin:
1. Visit http://localhost:3001
2. Login with `admin` / `admin`
3. See the order appear instantly
4. Click "Start Preparing"
5. Then "Mark Ready"
6. Finally "Mark Completed"
7. Watch the customer side update instantly!
